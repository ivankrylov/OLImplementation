 597 // Here are all the intrinsics known to the runtime and the CI.
 598 // Each intrinsic consists of a public enum name (like _hashCode),
 599 // followed by a specification of its klass, name, and signature:
 600 //    template(<id>,  <klass>,  <name>, <sig>, <FCODE>)
 601 //
 602 // If you add an intrinsic here, you must also define its name
 603 // and signature as members of the VM symbols.  The VM symbols for
 604 // the intrinsic name and signature may be defined above.
 605 //
 606 // Because the VM_SYMBOLS_DO macro makes reference to VM_INTRINSICS_DO,
 607 // you can also define an intrinsic's name and/or signature locally to the
 608 // intrinsic, if this makes sense.  (It often does make sense.)
 609 //
 610 // For example:
 611 //    do_intrinsic(_foo,  java_lang_Object,  foo_name, foo_signature, F_xx)

 611 // followed by a specification of its klass, name, and signature:
 612 //    template(<id>,  <klass>,  <name>, <sig>, <FCODE>)

 614 // If you add an intrinsic here, you must also define its name

 616 // the intrinsic name and signature may be defined above.

 618 // Because the VM_SYMBOLS_DO macro makes reference to VM_INTRINSICS_DO,

 620 // intrinsic, if this makes sense.  (It often does make sense.)

 623 //    do_intrinsic(_foo,  java_lang_Object,  foo_name, foo_signature, F_xx)
